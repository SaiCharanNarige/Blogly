#  Blogly – Full-Stack Social Media Web App

Blogly is a **full-stack social media platform** built with React and Node.js that lets users sign up, post tweets, follow others, and manage personal profiles — similar to Twitter.  
This project demonstrates complete frontend-backend integration, user authentication, and modern web-app architecture.

Blogly is designed to showcase:
- Scalable **React + Redux** frontend structure.
- RESTful API integration with an **Express/Node.js backend**.
- Authentication, tweet creation, likes, retweets, bookmarks, and profile management.
- Practical example of deploying a full-stack app using modern tools.

---

##  Features

-  **User Authentication** (Sign-up / Sign-in / Sign-out)  
-  **Tweet System** (Create, Edit, Delete, Like, Retweet, Reply)  
-  **Follow / Unfollow Users**  
-  **Bookmarks** for saving tweets  
-  **Profile Editing** with avatar upload  
-  **Responsive UI** built with React and CSS  

---

##  Tech Stack

| Layer | Technologies |
|--------|---------------|
| **Frontend** | React 17, Redux Toolkit, React Router DOM, Axios |
| **Backend** | Node.js, Express.js |
| **Database** | MongoDB (expected) |
| **Styling** | CSS3 |
| **Others** | React Dropzone Uploader, Emoji Picker, React Infinite Scroll |
